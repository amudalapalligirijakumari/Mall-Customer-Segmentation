PROJECT NAME:'MALL CUSTOMER SEGMENTATION'

## Project Scope

This project aims to segment mall customers into meaningful groups using Machine Learning techniques. By analyzing customer information such as annual income and spending score, the project identifies customers with similar purchasing behavior.

The project covers data loading, exploratory data analysis, visualization, cluster identification using the Elbow Method, and customer segmentation using K-Means Clustering. The generated customer groups can help businesses understand customer preferences and improve marketing strategies.

The scope of this project is limited to customer segmentation and behavior analysis. It does not include customer prediction, recommendation systems, or advanced marketing analytics. The project serves as a beginner-level machine learning application demonstrating the use of unsupervised learning in real-world business scenarios.
